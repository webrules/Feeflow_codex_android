# SectionHeader spec

**iOS source:** `Feedflow/Views/BookmarksView.swift`

## Purpose

Reusable uppercase section header for bookmark sections.

## Android UI

- Horizontal row.
- Caption-sized uppercase text in secondary color.
- Spacer fills remaining width.
- Top padding equivalent to iOS 8 pt.

## Behavior

- Pure presentation.

## Test cases

- Renders supplied title.
- Applies uppercase styling.
- Uses secondary text color.
